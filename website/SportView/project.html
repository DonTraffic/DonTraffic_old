О проекте

Работа выполнена для сдачи проекта в Вологодском коледже связи и информационных технологий


SportView - это сайт, с упражнениями для ознакомления и применения на практике для физического усиления


Подневная информация по созданию

[03.11.20]

Я начал создание с планировки дизайна. Дальше подготовка рабочей области, создание директории сайта (папок и файлов). Вёрстку начал с «шапки» сайта, там логотип, иконкой и меню.

[Завершено планирование]

[04.11.20]

Ниже идёт секция с мышечной анатомией человека и рядом рекомендации по тренировкам. Для создания этой секции нужны рекомендации и само изображение анатомии. Нашёл изображение анатомии, удалил фон в фотошопе. Было решено разделить мышечную анатомию человека на отдельные мышцы и выставлять их. Разделил мышцы в фотошопе и экспортировал в png.

[Завершено разделения картинки с анатомией]

[05.11.20]

Многочасовая подгонка стыков мышц. Завершён передний план анатомии мышц человека. Добавлен эффект при наведении на отдельную мышцу.

[Завершён блок с анатомией]

[07.11.20]

Начинаю вёрстку всплывающего блока с тренировками и видео. Блок пока без какого-либо функционала и с тестовым контентом

[Завершена тестовая вёрстка блока с тренировками]

[09.11.20]

Начало работы над функционалом блока тренировок. Переделывание блока тренировок, для улучшения функционала.

[10.11.20]

Работа над функционалом открытия и закрытия блока тренировок. Создание массива с контентом (ссылка на видео, информация по тренировкам)

[11.11.20]

Доработка дизайна.

[12.11.20]

Cоздание тестового функционала по замене видео ссылок и текста при нажатии на отдельную группу мышц.
