Oтрисовки списка элементов на основе массива данных

Для отрисовки, буду использовать код <li v-for="name in names"> {{ name }} </li>
Где names исходный массив, а name это ссылка на текущий элемент массива

  • {{ name }}

Так же для сложных массивов можно использовать :key='name.addition'

События

Для событий используеться вот такой синтаксис: v-on:событие

Компоненты

В Vue.js есть разделение на компоненты, они идут отдельным файлом (component.js)

Создать компонент можно с помощью синтаксиса:
Vue.component('text-one-comp', {
template: `<span>я <i>робот дурачёк</i></span>`,
})

А вставить соответственно тегом: <text-one-comp></text-one-comp>

Вывод --

Что бы вставить содержимое внутри тега в компонент, нужно указать тег <slot></slot> в компоненте:

html -- <text-two-comp>Привет, мир. Я программист (почти)</text-two-comp>
Компонент -- Vue.component('text-two-comp', {
template: `<span><i><slot></slot></i> </span>`,
})
Вывод -- Привет, мир. Я программист (почти)

Что бы взять данные из Vue и приминить из в компоненте, используеться props: ['name']

Для исользования в компоненте, используеться структура -- {{ name }}
